# 第3章 向量空间与子空间

## 3.5 四个子空间的维数

- 矩阵的秩是主元的个数
- 子空间的维数是基中向量的个数

**四个基本子空间：**

- $R^n$的一个子空间：行空间 $C(A^T)$
- $R^m$的一个子空间：列空间 $C(A)$
- $R^n$的一个子空间：零空间 $N(A)$
- $R^m$的一个子空间：左零空间 $N(A^T)$

行空间与列空间有相同的维数r，该数r就是矩阵的秩。

### 矩阵R的四个子空间

- 行空间的维数等于秩r，R的非零行构成行空间的一组基
- 列空间的维数等于秩r，主元列构成列空间的一组基
- 零空间的维数为 $n - r$，特殊解构成零空间的一组基
- 设 $m \times n$矩阵A的秩为r，则A左零空间的维数为 $m - r$
- 空间 $R^n$中，行空间与零空间的维数分别为r和 $n - r$
- 空间 $R^m$中，列空间和左零空间维数分别为r和 $m - r$

### 矩阵A的四个子空间

- A与R具有相同的行空间，两个行空间有相同的维数r以及相同的基。
	- 消元法改变了行，但没有改变行空间
- A的列空间的维数为r，列秩等于行秩
- A与R有相同的零空间，两个零空间有相同的维数 $n - r$和相同的基
- $(列空间的维数) + (零空间的维数) = R^n的维数$
- A的左零空间（即 $A^T$的零空间）的维数为 $m - r$

**线性代数基本定理（第一部分）**：对 $m \times n$矩阵A，列空间与行空间的维数均为秩r，两个零空间维数分别为 $n - r$和 $m - r$。

# 第4章 正交性

## 4.1 四个子空间的正交性

- 行空间垂直于零空间。A的每个行向量垂直于 $Ax = 0$的每个解。
- 列空间垂直于 $A^T$的零空间。

**定义**：正交子空间：同一向量空间的两个子空间V与W正交是指V中任意向量v与W中任意向量w垂直。

- 零空间 $N(A)$与行空间 $C(A^T)$是 $R^n$中的正交子空间
- $R^m$中，左零空间 $N(A^T)$与列空间 $C(A)$正交

### 正交补

**定义** 正交补：子空间V的正交补是指垂直于V的全体向量构成的子空间。V的这个正交子空间记作 $V^{\perp}$（读作“V垂”）

**线性代数基本定理（第二部分）** 对 $m \times n$矩阵A，

- 零空间 $N(A)$是行空间 $C(A^T)$的正交补（在 $R^n$中）
- 左零空间 $N(A^T)$是列空间 $C(A)$的正交补（在 $R^m$中）

正交补的意义在于每个向量x均可分解为一个行空间向量 $x_r$加上一个零空间向量 $x_n$。

- 如果向量v与自身正交，那么v是零向量

### 合并子空间的基

- $R^n$中任意n个线性无关向量一定生成 $R^n$，因此它们构成一组基；
- 生成 $R^n$的任意n个向量一定线性无关，因此它们构成一组基。

对于 $n \times n$方阵：

- 如果A的n个列向量线性无关，那么它们生成 $R^n$，因此 $Ax = b$是可解的
- 如果A的n个列向量生成 $R^n$，那么它们线性无关，因此 $Ax = b$有唯一解

## 4.2 投影

- 当将b投影到直线上时，它的投影p是b分解到该直线方向上的部分。
- 如果将b投影到平面上，投影p是b分解到该平面上的部分。
- 投影p等于Pb，即投影矩阵P左乘b等于p。

### 到直线上的投影

将b投影到a上的误差 $e = b - \hat{x}a$， $a \cdot (b - \hat{x}a) = 0$或 $a \cdot b - \hat{x} a \cdot a = 0$， $\hat{x} = \frac{a \cdot b}{a \cdot a} = \frac{a^T b}{a^T a}$。

b到过a的直线上的投影为向量 $p = \hat{x}a = \frac{a^T b}{a^T a}a$。

投影矩阵P： $p = a \hat{x} = a \frac{a^T b}{a^T a} = Pb$，此时 $P = \frac{aa^T}{a^T a}$

当P的作用是投影到一个子空间中时， $I - P$的作用是投影到其正交补子空间中。

### 到子空间上的投影

误差向量 $b - A \hat{x}$，它垂直于子空间哦，误差 $b - A \hat{x}$与所有基向量 $a_1, a_2, \cdots, a_n$均构成直角。

- $A^T (b - A \hat{x}) = 0$或正规方程： $A^T A \hat{x} = A^T b$。
- $\hat{x} = (A^T A)^{-1} A^T b$
- b在子空间上的投影p： $p = A \hat{x} = A (A^TA)^{-1} A^T b$
- 投影矩阵P： $P = A(A^T A)^{-1} A^T$


- （1）子空间是A的列空间
- （2）误差向量 $b - A \hat{x}$垂直于列空间
- （3）因此 $b - A \hat{x}$在 $A^T$的零空间（即A的左零空间）中，此即为 $A^T (b - A\hat{x}) = 0$

$A^T$的左零空间包含误差向量 $e = b - A \hat{x}$，而向量b分解为投影p与误差 $e = b - p$。投影则生成一个以p、e、b为边的直角三角形。

- $A^T A$可逆当且仅当A的各列线性无关。
- 当A各列线性无关时， $A^T A$是可逆的对称方阵。

## 4.4 标准正交基于格拉姆-施密特方法

**定义** 如果向量组 $q_1, q_2, \cdots, q_n$满足

$$
q_i^{T} q_j = 

\begin{cases}
0, \ \ i \neq j \ (正交向量) \\
1, \ \ i = j \ (单位向量：|||q_i = 1|)
\end{cases}
$$
则称 $q_1, q_2, \cdots, q_n$是标准正交向量组，这时，将各列分别为 $q_1, q_2, \cdots, q_n$的矩阵记为Q。

- $Q^TQ = I$：各列 $q_1, q_2, \cdots, q_n$标准正交
- 当Q为方阵时， $Q^TQ = I$意味着 $Q^T = Q^{-1}$，即转置等于逆
- 在方阵情形下，称Q为正交矩阵
- 每个置换矩阵都是正交矩阵

如果矩阵Q有标准正交列向量组 $(Q^TQ = I)$，那么它作用于向量时，保持向量的长度不变，即 $||Qx|| = ||x||$对任意的x均成立。

Q还保持两个向量内积不变： $(Qx)^T (Qy) = x^T Q^T Qy = x^T y$。

### 利用标准正交基投影：Q取代A

基向量恰好是标准正交的，则有：

- $\hat{x} = Q^Tb$
- $p = Q \hat{x} = QQ^T b$
- $P = QQ^T$

### 格拉姆-施密特方法

**生成标准正交向量组的“格拉姆-施密特方法”**

以三个线性无关向量 $a, b, c$为例，构造三个正交向量 $A, B, C$，生成三个标准正交向量 $q_1, q_2, q_3$：

- 首先取 $A = a$
- B必须垂直于A，基于b，减去b沿A方向的投影，就得到B： $B = b - \frac{A^T b}{A^T A}A$
- 第三个向量源自c，减去c在A和B两个方向上的分量，得到C： $C = c - \frac{A^Tc}{A^TA}A - \frac{B^Tc}{B^TB}B$
- 最后将得到的正交向量分别除以各自长度，即： $q_1 = \frac{A}{||A||}, q_2 = \frac{B}{||B||}, q_3 = \frac{C}{||C||}$

### 分解式 A=QR

- $A = QR$，关联矩阵R是三角形矩阵
- $A = QR$是紧凑版的格拉姆施密特方法，用 $Q^T$左乘该式就得到 $R = Q^TA$
- 格拉姆施密特方法： 由线性无关向量组 $a_1, a_2, \cdots, a_n$构造出标准正交向量组 $q_1, q_2, \cdots, q_n$。由上述列向量组构成的矩阵满足 $A = QR$。由于后续的 $q_j$与前面的 $a_i$正交（ $i < j$），所以 $R = Q^TA$是上三角形矩阵。

任意具有线性无关列向量组的 $m \times n$矩阵A均有分解式 $A = QR$，其中 $m \times n$矩阵Q各列标准正交，方阵R是上三角形矩阵，切其对角元素均为正数。

对于最小二乘法：

- $A^TA = (QR)^TQR = R^TQ^TQR = R^TR$
- 最小二乘方程： $A^TA \hat{x} = A^Tb$化简为： $R^TR \hat{x} = R^TQ^Tb$，可得： $R \hat{x} = Q^Tb$
- $\hat{x} = R^{-1}Q^Tb$

# 第5章 行列式

## 5.1 行列式的性质

- 方阵A的行列式是一个数字，记为 $\det A$或 $|A|$
- 当一个方阵不可逆时，它的行列式等于零
- A可逆时， $A^{-1}$的行列式是 $\frac{1}{\det A}$
- 当方阵的行线性相关时，一定会有 $\det A = 0$
- 主元的乘积就是行列式

n阶行列式能够通过三种不同的方法进行计算：

- （1）n个主元相乘（再乘1或-1），这是主元公式
- （2） $n!$个项相加（各项乘1或-1），这是基础公式
- （3）n个低一阶行列式的组合（各项乘1或-1），这是余子式公式

### 行列式的性质

- 性质1：n阶单位矩阵的行列式是1
	- $\det I = 1$，对应于单位立方体体积是1
- 性质2：互换两行时行列式改变正负号
- 性质3：行列式对单独的每一行是一个线性函数（其他行固定不变）
	- 若用数字t乘第1行元素，相当于用数字t乘行列式
	- 若第1行的每个元素都是两个数字之和，则行列式可拆分成两个行列式之和
- 性质4：若A中有两行元素对应相等，则 $\det A = 0$
	- 有两行元素对应相等的方阵A不可逆
- 性质5：从一行中减去另一行的倍数，行列式保持不变
	- 结论：从A到U的常规消元步骤（不换行）不改变方阵的行列式，因此 $\det A = \det U$
- 性质6：若方阵A的某一行的元素全为零，则 $\det A = 0$
- 性质7：若 $A = (a_{ij})_{n \times n}$是一个三角形矩阵，则 $\det A = a_{11}a_{22} \cdots a_{nn}$为主对角线上元素的乘积
- 性质8：若A是奇异矩阵，则 $\det A = 0$；若A是可逆矩阵，则 $\det A \neq 0$
- 性质9：AB的行列式是 $\det A$乘 $\det B$： $|AB| = |A||B|$
- 性质10：转置矩阵 $A^T$与A有相同的行列式

## 5.2 排列与代数余子式

### 主元公式

 消元法得到 $A=LU$，主元 $d_1, d_2, \cdots, d_n$位于上三角形矩阵U的主对角线上，若消元过程中没有换行，则将主元相乘即可计算行列式： $\det A = (\det L) (\det U) = 1 \cdot (d_1 d_2 \cdots d_n)$。

若考虑换行情况，则加入置换矩阵 $PA=LU$，其中置换矩阵P的行列式是-1或+1， $(\det P)(\det A) = (\det L)(\det U)$可得 $\det A = \pm (d_1 d_2 \cdots d_n)$。

每个主元都是行列式的比值，用行列式计算主元：第k个主元是 $d_k = \frac{d_1 d_2 \cdots d_k}{d_1 d_2 \cdots d_{k - 1}} = \frac{\det A_k}{\det A_{k - 1}}$

### 计算行列式的基础公式

$$
\begin{aligned}
\det A & = 全部n!个按列标排列 \alpha, \beta, \cdots, \omega 选取的简单行列式之和 \\
& = \sum (\det P) a_{1 \alpha} a_{2 \beta} \cdots a_{n \omega}
\end{aligned}
$$

### 利用代数余子式计算行列式

A的行列式是任意第i行元素与其他行所构成的代数余子式的点积，代数余子式公式： $\det A = a_{i1}C_{i1} + a_{i2} C_{i2} + \cdots + a_{in} C_{in}$.
每个代数余子式 $C_{ij}$（n-1阶，相比A划去了第i行和第j列）应包括正确的符号，代数余子式： $C_{ij} = (-1)^{i + j} det M_{ij}$。
